## filename

### filename Constraints

**pattern**: the string must match the following regular expression:&#x20;

```regexp
^[^<>:;,?"*|/]+\.(qual454|qual)(\.(gz|zip|rar|arj|tar|7z|bz2))?(\.gpg)?$
```
