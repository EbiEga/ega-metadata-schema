## filename

### filename Type

`string` ([Filename pattern of a MD5 file](ega-4-defs-check-filetype-checks-based-on-its-filename-anyof-md5-filename-patterncheck-properties-filename-pattern-of-a-md5-file.md))

### filename Constraints

**pattern**: the string must match the following regular expression:&#x20;

```regexp
^[^<>:;,?"*|/]+\.(md5|MD5)(\.(gz|zip|rar|arj|tar|7z|bz2))?(\.gpg)?$
```

[try pattern](https://regexr.com/?expression=%5E%5B%5E%3C%3E%3A%3B%2C%3F%22*%7C%2F%5D%2B%5C.\(md5%7CMD5\)\(%5C.\(gz%7Czip%7Crar%7Carj%7Ctar%7C7z%7Cbz2\)\)%3F\(%5C.gpg\)%3F%24 "try regular expression with regexr.com")

### filename Examples

```json
"my_file1.md5.gz.gpg"
```
